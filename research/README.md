# 研究结果库 (research/)

每个回测/测算跑完都在此存一条, 方便后续对比分析、避免重跑、追踪结论演变。

- `results.jsonl` — 机器可读, 一行一个实验 (id/date/topic/universe/method/metrics/conclusion/scripts/data/open_items)。新结果**追加**一行, 不覆盖。
- 本文件 — 人看的索引表, 加新实验时同步加一行。

约定: id = `YYYY-MM-DD_slug`; metrics 存关键数字(夏普/t/年化/胜率等); conclusion 一句话能复用的结论; 推翻旧结论时新开一条并在 conclusion 里注明推翻了哪条。
