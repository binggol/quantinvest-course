一、核心理念(先懂这个)
A股能稳定赚的只有四类,其余大半证伪:
  1. 避雷 — 剔除"市场还没反应的隐患"(造假DSRI/毛利恶化/热榜透支), 不是人人皆知的明雷(商誉/质押=已price-in)。
  2. 事件腿 — 有催化剂、可重复: 业绩抢跑、回购、纳入、海力士映射。
  3. 系统多因子 — 顾问Pro(切腿对冲)+ 挖矿精选因子(SOTA+XGB)。
  4. 纪律风控 — 买入后 −10%硬止损 + 吊灯止盈, 管的是风险不是收益。
成熟市场的价值/质量/突破/低吸/缩量回踩, 搬A股大半反向或失效(散户主导+壳价值+做空受限)。别追技术形态择点。
二、每天怎么操作(工作流)
  1. ① 开盘前: 打开 📋每日操作台 — 一屏看 底仓调仓 + 事件腿今日触发(抢跑/回购/海力士/打新) + 持仓踩雷 + ④数据新鲜度(🔴过期别用旧清单)。
  2. ② 下单: 按 组合清单 或事件腿清单买。看「日均成交」列, <2亿💧的票分批慢买别硬怼。下单后去 卖出提醒页 录入持仓, 选好「来源腿」。
  3. ③ 盘中(可选): 14:30 看 海力士映射(半导体做T信号); 持仓做T看 经典做T(成本线/止损线叠分时)。
  4. ④ 买入后风控: 每只持仓在 事件止损页 看两条线 — 🔴−10%跌破强平 / 🟢吊灯止盈跌破锁利。
  5. ⑤ 卖出: 卖出提醒页点💰卖出, 自动记台账。
  6. ⑥ 定期复盘: 实盘跟踪页 看各腿实盘 vs 回测对得上不(攒3-6个月)。

四、已验证结论(别重蹈覆辙)
这些"听起来很对"的策略, 数据证明在A股扣成本后没edge, 别再试:
  • 绿买红卖/红绿盘四象限择点 — 127万样本全象限净超额负(t−4~−24), "缩量回踩MA20低吸"=−0.20%/胜42%。
  • 突破/打板/杯柄形态 — 单笔骗局→组合−80%, 封板率仅10-40%。
  • 低吸高抛/日内方向预测 — 命中48.8%≈抛硬币, 分钟级超额被成本washout。
  • 分批止盈(+5%/+10%各卖) — 削峰降夏普(0.197→0.183), 不如单仓吊灯。
  • −5%/−3%硬止损 — 太紧被下影线插针洗, A股−10%收盘触发才最优。
  • 缠论/艾略特/威科夫 — 事后画图准、事前模糊, 形态在存量博弈=噪声。
共同教训: 择点/形态在A股是直觉陷阱。能赚的是催化剂(事件)+ 剔除隐患(避雷)+ 纪律(止损止盈), 不是"精准抄底逃顶"。
